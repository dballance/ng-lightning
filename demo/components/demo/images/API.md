# `<ngl-avatar>`

### Input
  * `src`: Path url.
  * `size?: 'x-small' | 'small' | 'medium' | 'large' = 'medium'`: Image size.
  * `type?: 'rectangle' | 'circle' = 'rectangle'`: Image shape.
  * `alt?`: Assistive text.

# `<figure[nglFigure]>`

### Input
  * `nglTitle?`: Image caption

### Content

  * `nglCrop?`: '16-by-9' | '4-by-3' | '1-by-1'`: Apply cropping to the specified element
